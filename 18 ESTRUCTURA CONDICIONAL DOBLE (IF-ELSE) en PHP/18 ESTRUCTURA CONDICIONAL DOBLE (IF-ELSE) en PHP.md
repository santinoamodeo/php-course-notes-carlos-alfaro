# 18 ESTRUCTURA CONDICIONAL DOBLE (IF-ELSE) en PHP
## Estructura solo PHP
```php
<?php 
    if(1>=9){
        echo 'Es verdadero';
    }else{
        echo 'No es verdadero';
    }
```

## Estructura PHP + HTML5
```php
    if(1>=4):
        echo 'Hola esto es verdadero';
    else: 
        echo 'Chau esto no es verdaero';
    endif;  
```
