# 05 Â¿Cuales son los TIPOS de DATOS en PHP?

## Booleanos (boolean)
```php
<?php
    echo TRUE;
    echo FALSE;
```

## Enteros (integer)
```php
    echo 10;
    echo -10;   
```

## Flotantes (double)
```php
    echo 7.1;
    echo 8585.32;
```
## Cadenas (string)
```php
    echo 'Hola mundo';
    echo "Chau mundo";
```

### Mostrar cual es el tipo de dato
```php
<?php
    var_dump(10);
    var_dump(-10);
    var_dump(TRUE);
    var_dump(FALSE);
    var_dump(1.80);
    var_dump(88888.55);
    var_dump('Hola mundo')
    var_dump("Adios mundo")
```
