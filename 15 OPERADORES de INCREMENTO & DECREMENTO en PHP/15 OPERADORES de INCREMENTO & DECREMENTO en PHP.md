# 15 OPERADORES de INCREMENTO & DECREMENTO en PHP

## Pre incremento ++
Realiza primero el incremento, y luego la accion solicitada, por ejemplo un echo

```php
    $numero = 10;
    echo ++$numero;
```


## Post incremento ++
Realiza primero la accion solicitada, por eemplo el echo, y despues incrementa

```php
    $numero = 10;
    echo $numero++;
```


## Pre Decremento --
```php
    $numero = 10;
    echo $numero--;
```


## Post decremento --
```php
    $numero = 10;
    echo $numero--;
```
